# Raiz Insurance Mod

Sistema avançado de seguro veicular para DayZ.

Desenvolvido originalmente para o servidor **Raiz PvE**, permitindo que jogadores protejam seus veículos através de um sistema de apólices, restauração e gerenciamento administrativo.

## Recursos

* InsuranceID único
* VehicleUID único
* Registro persistente em JSON
* Sistema de monitoramento de veículos
* Restauração automática
* Planos BASIC, PLUS e VIP
* Painel administrativo
* Controle por SteamID
* Compatibilidade com veículos vanilla e modded
* Configurações externas

## Planos

### BASIC

* 1 restauração

### PLUS

* 3 restaurações

### VIP

* Restaurações ilimitadas

## Arquivos de Configuração

```txt
profiles/RaizInsuranceData/
```

* InsuranceConfig.json
* VipConfig.json
* AdminConfig.json
* insured_vehicles.json
